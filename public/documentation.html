              {

              }
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/[genre title] GET none A JSON object holding data about a genre 
            
              {
                genre: "abc",
                description: "abc"
              }
Return data about a director by name /directors/[name] GET none A JSON object holding data about a director including bio, birth year, death year 
            
               {
                 director: "abc",
                 bio: "abc",
                 birthyear: 1955,
                 deathyear: 2012
               }
Allow new users to register /users POST A JSON object holding data about the new user including username and mail. Format: 
            
            {
              name: "abc",
              mail: "mail@example.com"
            }
A JSON object holding data about the new user including ID, username and mail. Format: 
            
            {
              id: 123,
              name: "abc",
              mail: "mail@example.com"
            }
Allow users to update their user info (username) /users/[ID]/[new username] PUT none A text message indicating that the username was successfully changed
Allow users to add a movie to their list of favorites /users/[ID]/movies/[movie title] PUT none A text message indicating that the movie was added to their list of favorites movies
Allow users to remove a movie from their list of favorites /users/[ID]/movies/[movie title] DELETE none A text message indicating that the movie was removed from their list of favorites movies
Allow existing users to deregister /users/[ID] DELETE none A text message indicating that the user email was removed
